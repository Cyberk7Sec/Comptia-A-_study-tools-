‚ö° 24-Pin ATX Power Connector

  • Location: Right edge of motherboard (largest power connector)
  • Purpose: Main power delivery to entire motherboard
  • Configuration: Usually 20+4 pin (can separate into 20-pin + 4-pin)
  • Critical: ALWAYS required - PC won't boot without it
  • Connection: From PSU directly to motherboard
üí° Quick Tip: This is the FIRST connector you should check if PC has no power. Make sure it's fully seated and clicks in place!

‚ö° 4/8-Pin EPS CPU Power

  • Location: Top-left corner, near CPU socket
  • Purpose: Dedicated power for CPU
  • Configuration: 4-pin, 4+4-pin (8-pin), or 8-pin
  • Critical: PC may power on but won't boot without this
  • Variants: Budget boards = 4-pin | Gaming/High-end = 8-pin or dual 8-pin
üí° Quick Tip: Don't confuse with PCIe power! EPS goes to the MOTHERBOARD near CPU, PCIe power goes to the GRAPHICS CARD.

üñ•Ô∏è CPU Socket

  • Location: Upper-left area, near EPS power connector
  • Purpose: Houses the CPU (processor)
  • Intel Types: LGA 1151, LGA 1200, LGA 1700
  • AMD Types: AM4, AM5, TR4/sTRX4 (Threadripper)
  • Critical Rule: CPU MUST match socket type exactly!
üí° Quick Tip: LGA (Intel) = pins on motherboard | PGA (older AMD) = pins on CPU. Handle with extreme care!

üíæ RAM Slots (DIMM)

  • Location: Right of CPU socket (vertical slots)
  • Typical Count: 2 or 4 slots
  • Type: DDR4 or DDR5 (NOT interchangeable!)
  • Best Practice: For 2 sticks, use slots 2 & 4 (counting from CPU)
  • Dual Channel: Matching pairs in slots 2&4 or 1&3 for best performance
üí° Quick Tip: "Skip One, Use Two" - Install RAM in slots 2 & 4 first (usually color-coded). Check your manual for preferred slots!

üíæ M.2 NVMe Slots

  • Location: Usually 2-3 slots scattered on board (one often near CPU, one near bottom)
  • Purpose: High-speed SSD storage (no cables needed!)
  • Types: M.2 NVMe (PCIe - FAST) or M.2 SATA (slower)
  • Installation: Insert at 30¬∞ angle, push down, secure with screw
  • Speed: PCIe Gen3 = ~3,500 MB/s | Gen4 = ~7,000 MB/s
üí° Quick Tip: M.2 drives get HOT! Many boards include heatsinks. Some M.2 slots share lanes with SATA ports - check manual!

üíæ SATA Ports

  • Location: Right edge or bottom-right (usually 4-6 ports)
  • Purpose: Connect SATA SSDs and HDDs
  • Speed: SATA III = 6 Gbps (600 MB/s max)
  • Remember: SATA drives need TWO cables (data + power)
  • Numbering: Usually labeled SATA0-5 or SATA1-6
üí° Quick Tip: SATA = 2 cables! Data cable (small L-shape) to motherboard + Power cable (wide L-shape) from PSU. If either is missing, drive won't work!

üéÆ PCIe x16 Slots (GPU Slots)

  • Location: Long horizontal slots below RAM area
  • Purpose: Graphics cards (primary use), high-speed expansion cards
  • Length: ~7 inches (longest slots on motherboard)
  • Best Slot: Top x16 slot usually has most lanes (x16 full speed)
  • Second Slot: May run at x8 or x4 speed (check manual)
üí° Quick Tip: Always use the TOP x16 slot for your main GPU! It has direct connection to CPU for best performance. High-end GPUs need 6/8-pin PCIe power cables!

üîå PCIe x1 Slots

  • Location: Short slots between x16 slots
  • Purpose: Sound cards, Wi-Fi cards, USB expansion, capture cards
  • Length: ~1 inch (very short)
  • Speed: PCIe 3.0 x1 = ~1 GB/s (enough for most add-on cards)
  • Compatibility: Can accept x1, smaller cards
üí° Quick Tip: These short slots are for adding functionality - Wi-Fi, better sound, more USB ports. NOT for graphics cards!

üîò F_PANEL (Front Panel Connectors)

  • Location: Bottom-left corner (small pins)
  • Purpose: Connects case power button, reset button, LED lights
  • Connections: POWER SW, RESET SW, POWER LED+/-, HDD LED
  • Critical: PC won't turn on if power switch not connected!
  • Tricky: Small pins, often labeled, need careful connection
üí° Quick Tip: Check your motherboard manual for exact pin layout! POWER SW is the most critical - without it, your power button does nothing!

üîå F_USB (Front USB Headers)

  • Location: Bottom edge (multiple headers)
  • Purpose: Connects case front panel USB ports to motherboard
  • Types: USB 2.0 (9-pin) and USB 3.0 (19/20-pin) headers
  • Critical: Front USB ports won't work without these!
  • Variants: USB 3.1/3.2 Gen 2 (USB-C) uses different header
üí° Quick Tip: If front panel USB doesn't work, check if cable is connected to motherboard header. USB 3.0 headers are keyed - can only go in one way!

üéµ F_AUDIO (Front Audio Header)

  • Location: Bottom edge, near USB headers
  • Purpose: Connects case front panel headphone/mic jacks
  • Type: HD Audio (Intel standard) - 9-pin connector
  • Critical: Front audio jacks won't work without this
  • Alternative: AC'97 (older standard, rarely used now)
üí° Quick Tip: Modern boards use HD Audio standard. If front headphone jack doesn't work, check this connection first!

üåÄ Fan Headers

  • CPU_FAN Location: Near CPU socket (REQUIRED for CPU cooler)
  • SYS_FAN Location: Scattered around board (case fans)
  • Type: 3-pin (voltage control) or 4-pin PWM (speed control)
  • Critical: CPU_FAN MUST be connected or PC may not boot!
  • Count: 3-8 headers typical (1 CPU, rest for case fans)
üí° Quick Tip: ALWAYS connect CPU cooler to CPU_FAN header! Board monitors this and will warn/shut down if not detected. 4-pin PWM fans allow better speed control!

üñ•Ô∏è Chipset

  • Location: Center of board (usually has heatsink)
  • Purpose: Controller for I/O, SATA, USB, PCIe lanes
  • Determines: # of SATA, USB ports, PCIe lanes, RAM support
  • Examples: Intel Z790, B760 | AMD X670, B650
  • Tiers: Z/X = enthusiast | B = mainstream | H/A = budget
üí° Quick Tip: Chipset determines motherboard features! Z/X series support overclocking, B series don't. This affects CPU and RAM overclocking capability!

üîå I/O Shield / Rear I/O Ports

  • Location: Back of motherboard (external ports)
  • Typical Ports: USB, Ethernet, Audio jacks, Video outputs (if CPU has iGPU)
  • USB Ports: Mix of USB 2.0 (black) and USB 3.0+ (blue/red/teal)
  • Network: RJ45 Ethernet (8-pin, usually with LEDs)
  • Video: HDMI, DisplayPort (only if CPU has integrated graphics)
üí° Quick Tip: Don't forget to install the I/O shield in the case FIRST before mounting motherboard! USB port colors: Black = USB 2.0, Blue = USB 3.0, Red/Teal = USB 3.1/3.2
